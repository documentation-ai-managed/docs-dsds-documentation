## Overview

dsds provides a centralized platform where you create, organize, and collaborate on project documentation. Build structured spaces for APIs, user guides, and internal wikis with version control, search, and real-time editing. Customize navigation and permissions to fit your team's workflow.

## Key Benefits for Teams

You gain efficiency with dsds's intuitive interface. Teams reduce documentation silos by centralizing knowledge. Features like templates speed up creation, while integrations connect to tools such as GitHub and Slack.

## Quick Start

Follow these steps to set up your first documentation space.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at https://dsds.example.com/register with your email.
  </Step>
  <Step title="Build First Space" icon="folder-plus">
    Click "New Space" and name it, such as "API Docs".
  </Step>
  <Step title="Add Content" icon="file-plus">
    Create pages using Markdown or MDX. Use components like `<Callout>` for highlights.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Embed dsds docs in your app
import { DSDSViewer } from '@dsds/viewer';

<DSDSViewer spaceId="your-space-id" apiKey="YOUR_API_KEY" />
```
```python
# Python SDK example
from dsds import Client

client = Client(api_key="YOUR_API_KEY")
space = client.spaces.get("your-space-id")
print(space.pages)
```
</CodeGroup>

## Navigation and Structure

dsds uses a hierarchical structure: Spaces contain Pages, which support nested subpages. Access via sidebar or search bar.

## Frequently Asked Questions
